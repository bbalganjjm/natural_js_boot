트리+그리드(CRUD)

  • 트리(N.tree)와 그리드(N.grid)로 구성된 예제입니다.
  • N.tree의 onSelect 이벤트 옵션을 통해 N.grid 컴포넌트를 연동하는 방법을 알 수 있습니다.
  • 공통코드 데이터를 자동으로 조회한 후 N.select 컴포넌트를 초기화하고 코드 데이터를 바인딩합니다.
  • 그리드(N.grid)로 입력/조회/수정/삭제를 할 수 있습니다.
