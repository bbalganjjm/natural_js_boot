버튼+이벤트(N.button)

Button N.button 컴포넌트를 이용하여 버튼을 생성할 수 있습니다.

N.cont object에 e.{id}.{event}를 선언

Event Button N.cont object에 e.{id}.{event}를 선언하면 N.button 컴포넌트 초기화와 jQuery 이벤트 바인딩이 동시에 처리됩니다.

버튼 옵션

Button Option N.button 컴포넌트의 옵션을 버튼(a, input[type=button], button) 태그의 data-opts 속성으로 선언할 수 있습니다.
